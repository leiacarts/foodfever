## Overview

FOOD FEVER is a restaurant review app that helps narrow down whatâ€™s good in the neighborhood. Featuring filters for locations and cuisines, this responsive app utilizes ES6 JavaScript and caches to function both online and offline.

### Learning Obectives

To apply accessibility to a responsive web application, and to create a service worker to cache requests, making the app accessible without a network connection.

### Features

* ARIA labels and roles
* Accessible images
* Caches
* Object Notation
* ES6

## Installation

Download repo. Using the command `python -m SimpleHTTPServer 8000` in terminal, run the localhost while inside the project folder.
